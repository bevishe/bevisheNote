# 1.前端html向后台传值

## 1.html通过url向后台传值

前端HTML：

```html
<a class="btn btn-sm btn-primary" th:href="@{/emp/}+${emp.id}">编辑</a>
<button th:attr="del_uri=@{/emp/}+${emp.id}" class="btn btn-sm btn-danger deleteBtn">删除</button>
```

Controller：

```java
    // 点击编辑跳转到修改页面
    @RequestMapping(value="/emp/{emp.id}")
    public String toUpdateEmp(@PathVariable("emp.id") String empId,ModelMap modelMap){
        Employee employee = employeeDao.get(Integer.valueOf(empId));
        Collection<Department> depts = departmentDao.getDepartments();
        modelMap.addAttribute("depts",depts);
        modelMap.addAttribute("emp",employee);
        return "emp/add";
    }
```

在前端页面将数据放在url时，需要使用${yourdata}，在后端获取变量是现在@RequestMap(value="/emp/{yourdata}") 只需要使用{yourdata}即可，同时在参数列表里面需要使用@PathVariable（"yourdata"）进行绑定。

# 2.后台向前端html页面传值

## 1.使用ModelMap的方法

对于前端html需要的变量，在后端Controller中将变量以kv的方法封装到ModeMap中进行传递，会发送到对应返回的页面。

前端需要emp和depts：

```html
<input type="hidden" name="_method" value="put" th:if="${emp!=null}"/>
						<input type="hidden" name="id" th:if="${emp!=null}" th:value="${emp.id}">
<!--						<input name="id" />-->
						<div class="form-group">
							<label>LastName</label>
							<input name="lastName" type="text" class="form-control" placeholder="zhangsan" th:value="${emp!=null}?${emp.lastName}">
						</div>
						<div class="form-group">
							<label>Email</label>
							<input name="email" type="email" class="form-control" placeholder="zhangsan@atguigu.com" th:value="${emp!=null}?${emp.email}">
						</div>
						<div class="form-group">
							<label>Gender</label><br/>
							<div class="form-check form-check-inline">
								<input class="form-check-input" type="radio" name="gender" value="1" th:checked="${emp!=null}?${emp.gender==1}">
								<label class="form-check-label">男</label>
							</div>
							<div class="form-check form-check-inline">
								<input class="form-check-input" type="radio" name="gender" value="0" th:checked="${emp!=null}?${emp.gender==0}">
								<label class="form-check-label">女</label>
							</div>
						</div>
						<div class="form-group">
							<label>department</label>
							<!--提交的是部门的id-->
							<select class="form-control" name="department.id">
								<option th:selected="${emp!=null}?${dept.id == emp.department.id}" th:value="${dept.id}" th:each="dept:${depts}" th:text="${dept.departmentName}">1</option>
							</select>
						</div>
						<div class="form-group">
							<label>Birth</label>
							<input name="birth" type="text" class="form-control" placeholder="zhangsan" th:value="${emp!=null}?${#dates.format(emp.birth, 'yyyy-MM-dd HH:mm')}">
						</div>
```

Controller:

```java
    // 跳转到添加页面
    @GetMapping(value = "/templates/emp")
    public String toAddPage(ModelMap model){
        Collection<Department> depts = departmentDao.getDepartments();
        model.addAttribute("depts",depts);
        model.addAttribute("emp",null);
        return "emp/add";
    }
```
